## 📦 업로드 가능 용량 안내

- GitHub 웹 업로드 최대 용량: **100MB 이하 / 파일당**
- 저장소 전체 권장 용량: **1GB 이하**
- 100MB 초과 파일(.exe, .msi 등 실행파일)은 **웹 업로드 불가**
- 대용량 파일은 Git LFS 사용 가능(최대 2GB/파일)
